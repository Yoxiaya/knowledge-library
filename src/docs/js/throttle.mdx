# 节流（Throttle）

## React Demo

界面包含两个区域，分别用于展示时间戳版和定时器版节流效果。

移动鼠标区域时，高频触发事件，节流函数会按照设定的 delay 限制触发次数。

## 什么是节流？

**目标：即使事件持续触发，也要限制执行频率。**

节流用于限制高频事件的触发次数，在一定时间内只执行一次函数。

常用于：

- scroll 滚动事件
- resize 窗口变化
- mousemove / mouseover
- 拖拽事件

---

## 两种常见实现方式

### 1. 时间戳版（leading）

特点：

- 立即执行第一次
- 下一次执行需要等待 delay

### 2. 定时器版（trailing）

特点：

- 不会立即执行第一次
- 最后一次会执行

---

## 对比

| 实现方式 | 是否立即执行 | 是否执行最后一次 |
| -------- | ------------ | ---------------- |
| 时间戳版 | ✔ 是         | ✖ 否             |
| 定时器版 | ✖ 否         | ✔ 是             |
