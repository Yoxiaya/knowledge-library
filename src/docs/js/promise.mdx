# Promise 基础案例

## Promise 是什么？

Promise 是一个用于处理异步的对象，有三个状态：

- pending：进行中
- fulfilled：成功
- rejected：失败

---

## Promise 常见用法

### 1. then / catch / finally

### 2. Promise.all / race / allSettled / any

### 3. async / await（语法糖）

# 手写 Promise 简化版

手写 Promise 核心点：

1. 维护三个状态：pending / fulfilled / rejected
2. 状态只能变一次
3. then 需要支持 pending 阶段的回调收集
4. 触发 resolve/reject 时执行对应回调队列

本项目内提供了一个简化版的 MyPromise 实现，用于理解原理。
